=====

Description
============
The traditional 'PONG' game re-created in JAVA.

Instructions
============ 
Player 01 controls: 
Up = 'w' key
Down = 's' key

Player 02 controls: 
Up = 'Up arrow'
Down = 'Down arrow'

A self driven project chosen as a way to learn about game development in JAVA.
